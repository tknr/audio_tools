# audio_tools

## convert_to_mp3

### what is this?
converts wav,flac,aiff,ogg,mpc to mp3.

### usage

bash convert_to_mp3.sh <folder>

## upconvert_mp3

### what is this?

upconverts bitrate of mp3 under all folders you suddgest.

### usage

bash upconvert_mp3.sh <target_folder>
